## 📌 项目简介

本项目旨在解决无创产前检测（NIPT）中检测时机选择与女胎异常精准识别的临床科学问题。基于临床检测数据，本项目构建了**“规律发现——时机推荐——精准预测——风险预警”**的四阶段递进式优化模型体系，为 NIPT 的个性化时机推荐提供了有力的科学依据。

本仓库收录了我们团队在此次竞赛中提交的完整学术论文及赛题原始文件。

## 🧠 核心模型与方法

本项目针对赛题的四个问题，分别应用了以下数学建模方法：

1. **非线性动态建模 (问题一)**：
   - 剔除异常值并进行数据清洗。
   - 采用 **Pearson 相关性分析**发现变量间弱相关性。
   - 构建 **广义相加混合模型 (GAMM)**，引入非线性光滑函数精准刻画 Y 染色体浓度随孕周和 BMI 的动态变化规律 ($R^2 > 0.75$)。
2. **多约束时点优化 (问题二)**：
   - 构建包含达标概率函数与分段风险函数的**期望风险量化体系**。
   - 采用**动态规划与全局优化策略**，在 7 个临床约束下划分最优 BMI 区间。
   - 引入**蒙特卡洛误差分析**验证模型鲁棒性（±5%误差下风险波动率 < 3.2%）。
3. **高维协变量修正模型 (问题三)**：
   - 采用 **Spearman 秩相关分析**从多维数据中降维，筛选出三大核心变量（X染色体浓度、体重、18号染色体Z值）。
   - 构建扩展维度的个体化 GAMM 预测模型，进一步压缩临床诊断风险值。
4. **异常预警分类器 (问题四)**：
   - 构建有监督机器学习框架，对比逻辑回归、SVM、KNN、决策树与**随机森林**。
   - 基于随机森林构建失败风险预警系统（AUC = 0.8909）。
   - 实施 **ROC 曲线阈值优化**，实现漏诊与误诊的最佳权衡（召回率提升至 84%）。
